# yandex backend school

Used frameworks and technologies:
* Flask
* sqlitedict
* docker

To run the service:
```
> cd yandex-backend-school/
> docker-compose build
> docker-compose up
```

To run tests:
```
> conda activate
> cd yandex-backend-school/
> cd Tests/
> pytest
```
